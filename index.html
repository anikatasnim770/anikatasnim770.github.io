About Me

Engineer, mentor, and silicon problem solver.

Hi, I'm Anika — Senior Design Verification Engineer at Ulkasemi Pvt. Limited in Dhaka, Bangladesh. I've spent four years building UVM testbenches for LPDDR memory interfaces and protocol IPs. I've learned that the most valuable thing in verification isn't knowing which tool to run — it's knowing exactly what the waveform should look like before the simulation starts.

My work spans test-plan development, UVM component design, UVM RAL integration, functional and code coverage closure, SVA assertion integration, and regression automation with Bash. I debug in Synopsys Verdi nWave, sign off coverage in Cadence ICCR, and work closely with RTL teams to iterate until the IP is clean.

Beyond verification, I mentor 15+ engineers on our team and run the Ulkasemi DV Vlog — a verification knowledge platform for early-career engineers across South Asia. I believe the Bangladesh semiconductor ecosystem deserves a global stage, and I'm working to help build it.
